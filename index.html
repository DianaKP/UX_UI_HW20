Project Overview UI Analysis Information Architecture & UI Navigation Responsive Design & Conclusion

Project Overview

The scope of this project was to find a government agency website that could use a re-design, focusing on design in general, navigation/information architecture, and accessibility for the vision impaired. I chose the Environmental Protection Agency, one - because environmental protection is near and dear to my heart, and two - because upon first glances on the site, I saw opportunity for some updated design, as well as many areas for better accessibility via color palette, and many navigational functions I felt could be simplified. So, without further ado, read on to see how I completed this case study!

The Problem

Users have trouble navigating EPA.gov using the current navigation and design is not accessible, color-wise to the vision impaired, or welcoming to average users.

The Solution

Analyze and re-organize information on the site to make more sense, and update the design to be more welcoming, accessible and user friendly.
